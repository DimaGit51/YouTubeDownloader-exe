Здравствуйте, мои маленькие любители программизма

На GitHub:
1. В репозитории нажимаете на зеленую кнопку "<>Code"
2. В открывшемся окне выбираете "Download ZIP" 
3. У вас скачивается архив
4. Когда он скачается, распокуйте его

После распаковки архива:
1. Откройти папку "YouTube Downloader_V<version>"
2. Запустите приложение exe "YouTube Downloader_V<version>.exe"
3. Скопируйте нужную ссылку на видео с ютуба и вставьте её в поле под текстом "Введите ссылку на видео YouTube"
4. Нажмите кнопку скачать
5. Загрузка видео пойдет в папку "Downloaded=video"
6. Если возникнет какая-то ошибка, то появится окно ошибки. Загрузка видео в папку прекратится
